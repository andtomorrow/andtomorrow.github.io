6. ‘switch’ 문

  • ‘switch’문은 여러 개의 if 문을 대신할 수 있음
  • 값 비교에 일치 연산자1 ===를 사용함
let age = prompt('Your age?', 18);

switch (age) {
  case 18:
    alert("Won't work"); // 주의. prompt 함수는 항상 숫자가 아닌 문자열을 반환함
    break;

  case "18":
    alert("This works!");
    break;

  default:
    alert("Any value not equal to one above");
}

  1. strict equality ↩︎
